--- CadAppData ---

FCompileExtDataEvent - Событие обновления данных

FOnClear - Событие срабатывает при необходимости очистить данные

FOnImportDataFromXls - Событие срабатывает при импорте данных из Xls файла

FOnSetPosition - Событие срабатывает при необходимости в таблице с данными установить указатель на требуемый элемент
        Obj - таблица - номер таблицы (0-Branchs, 1-Nodes)
        Data - номер элемента (ветви/узла)

DrawWin_IsClosing1 - True, если окно отрисовки находится в режиме закрытия

ViewMode - Вид отображения схемы

PaintBoxCanvas - Основной элемент отрисовки графики

TablData_IsModyfid - Интерфейс к внешним данным
